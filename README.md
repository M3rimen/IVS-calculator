Spusteni
--------

Instalace:
- ~/cd install/
- ~/sudo dpkg -i ivs-calculator-1.0.0-Linux.deb
- ~/sudo dpkg -i ivs-stddev-1.0.0-Linux.deb

Spusteni:
- ~/calculator
- ~/stddev

Odinstalace:
- ~/sudo apt remove ivs-calculator
- ~/sudo apt remove ivs-stddev
